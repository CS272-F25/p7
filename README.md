
# Animal Learning Website

## Overview

This is an online educational website designed as a web development course project. The website aims to make learning about animals engaging and accessible for users of all ages. It focuses on providing a personalized and interactive learning experience, allowing users to explore animal information and test their knowledge through quizzes.

## Features

* **Interactive Learning:** Users can access detailed information about various animals.
* **Personalized Experience:** User progress and preferences are stored locally using the browser's Local Storage.
* **Quizzes:** Interactive quizzes allow users to test their knowledge and track performance.
* **Data Management:** Animal data is stored in JSON files, ensuring structured and easily maintainable content.
* **Clean Design:** The website is designed to be simple, clear, and free of unnecessary distractions to enhance the learning experience.

## Technologies Used

* HTML/CSS
* JavaScript (Vanilla JS)
* JSON (for data storage)
* Local Storage (for user-specific data)
